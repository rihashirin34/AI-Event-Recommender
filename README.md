Harmoniq – AI Event Recommendations

Harmoniq is a web-based event recommendation platform that uses AI-inspired scoring to suggest events tailored to users’ interests and vibes. Users can select their favorite categories and mood, and the platform generates personalized recommendations with match scores.

Features

AI-style recommendations: Events are scored based on user-selected interests and vibes.

Interactive onboarding: Users pick interests and vibes in a friendly UI.

Event cards: Visual, animated, and responsive event display.

Modal details: Click an event to see full info and “Book Ticket” button.

Modern design: Glassmorphism, gradient texts, and responsive layout.

Standalone frontend: Fully functional without a backend (mock data included).

Tech Stack

Frontend: HTML5, CSS3, Vanilla JavaScript

Styling: Glassmorphism, gradients, responsive design

No backend required (mock events are embedded in JS for demo)

Optional Backend: Can be integrated with Node.js + MongoDB for persistent data

Getting Started
1. Clone Repository
git clone <your-repo-url>
cd harmoniq

2. Open in Browser

Simply open index.html in your browser (Chrome/Firefox recommended).

3. Usage

Click “Find Events” on the landing page.

Select your interests and vibes.

View top recommended events on the dashboard.

Click an event to see detailed info and book ticket.
